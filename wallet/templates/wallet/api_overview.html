Wallet API Overview

This page provides a brief overview of the available API endpoints for the Wallet Application.

Endpoints

  • Create Wallet: POST /api/wallets/
    Creates a new wallet for a user.
  • Create Transaction: POST /api/wallets/<wallet_id>/create-transaction/
    Initiates a pending transaction for credit, debit, or transfer.
  • Create Transfer Transaction: POST /api/wallets/transfer/create/
    Creates a pending transfer transaction between two wallets.
  • Process Transaction: POST /api/transactions/<transaction_id>/process/
    Confirms and finalizes a pending transaction, updating wallet balances.
  • Check Wallet Balance: POST /api/wallets/balance/
    Retrieves the balance of a wallet using the private key for authentication.

Instructions

Each endpoint requires specific parameters in JSON format. Refer to each method's documentation for details.
